Quick Start

Quick Start

Main points.

The Components.

A component defines a piece of user interface that you can customise. It comes with his own logic. It can be a whole part, like an header, or a smaller piece, like a menu or a set of buttons, an image, a map… You can find all the available components in the Components Catalog.

The Views.

The view is made of a set of components. You can add a new component to a view by drag and drop. The interesting part of the view is that it is a reusable piece of interface that you can use and reuse in many pages (or into a modal panel).

The Pages.

A page is made of one or more views. This is the HTML page as such, that will be loaded in the browser to show the content to your visitors.
