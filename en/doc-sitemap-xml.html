Robots Sitemap xml

Generate an xml sitemap file

SledPlus only.

Hello

In order to improve your SEO, it’s better to have a sitemap xml file that describes your pages for the search engines.

Basicaly, you have one single thing to do with SledPlus :

  • Go to the settings side menu
  • Enter your domain name : http://yourdomain.com


That’s it ! Publish your project and you will see a newly created sitemap.xml allong with a robot.txt file that is nesessary for the robots to localize your sitemap file.
