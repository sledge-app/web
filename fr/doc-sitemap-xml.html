Sitemap xml pour les robots

Générer un fichier xml de sitemap.

Seulement avec SledPlus.

Salut

Afin d’optimiser votre SEO il est préférable de générer à la racine de votre projet un fichier XML de sitemap. Sled le fait pour vous très simplement:

  • Allez dans les paramètres du projet.
  • Ajouter le nom de domaine de votre site (http://votre domaine.com).

C’est tout ! Vous pouvez à présent publier votre projet, un nouveau fichier sitemap.xml apparaitra, avec un fichier robots.txt, nécéssaire pour indiquer aux robots l’emplacement de notre XML.
