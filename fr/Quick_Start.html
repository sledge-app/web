Les composants

Un composant définit un fragment d’interface. C’est le plus petit sous ensemble logique que Sledge propose de manipuler. Il peut s’agir d’un entête de page qui se configure (fond, menu, position des titres), d’une image, d’un carrousel, d’une carte… Vous trouverez tous les composants disponibles dans le catalogue de composants.

Les vues

Une vue est faite d’un ensemble de composants. Vous ajoutez des composants dans une vue par glisser-déposer. L’intérêt de la vue est de constituer une unité d’interface ré-utilisable dans différents contextes (pages, fenêtre modal).

Les pages

Une page, vous pouvez en créer plusieurs avec Sledge, est faite de une ou plusieurs vues. C’est la page html en tant que telle, celle qui sera chargée et vue par le visiteur de votre site.
