Paramètres de page

Paramètres de page

La Page représente la page HTML en tant que telle, celle qui sera utilisée par votre visiteur pour découvrir votre contenu. Si vous avez une page bonjour, suite à la publication, votre page sera disponible sur /bonjour.html, ou après déploiement sur votre serveur, monNomDeDomaine.com/bonjour.html.

Il existe un certain nombre de paramètres associés à la page. Vous les trouvez en sélectionnant une page dans le menu de gauche, puis dans la zone de droite, surlignée en rouge dans l’illustration ci-contre.


Onglet « Paramètres généraux »

Type de page.

Définit la mise en forme générale de la page, voir la documentation “paramètres de page”.


Nom de la page.

C’est le nom que vous avez défini, il apparait dans le menu latéral. Ce nom sera utilisé à la publication pour le nom de la page html. Il sera aussi utilisé dans les divers menus qui peuvent être utlisés par des composants, à moins que vous le localisiez (voir ci-dessous).


Nom localisé de la page.

Ce nom apparaitra dans la version adaptée à la langue dans les menus et dans les onglets du navigateur.



Mot-clés (localisable)

Listez ici les mots-clés qui définissent le mieux votre page. Sépérez-les par une virgule. Cela sera utile pour le référencement sur les moteurs de recherche.


Description (localisable)

Une discription de votre site, peut être utilisée par les moteurs de recherche.


Auteur

Votre nom ou une référence de votre choix, pour les moteurs de recherche.


Contenu en pleine largeur

Permet à votre contenu de s’étendre à l’infini sur toute la largeur du navigateur. L’option non cochée, votre contenu s’élargira jusqu’à une certaine largeur, puis se fixera au centre de la page au delà.


Onglet « Arrière plan »

Permet de définir un fond pour la totalité de la page. Il peut s’agir d’une couleur, d’un motif ou d’une image. Ce fond sera recouvert par les fonds définis au niveau des vues ou des composants.

Onglet « Scripts »

Seulement avec SledPlus.

Salut

Vous pouvez ici ajouter : 


  1. - Un script javascript qui se lancera avant le chargement complet de la page.
  2. - Un script javascript qui se lancera après le chargement complet de la page. Il sera encapsulé dans un événement DOMContentLoaded.
  3. - Du contenu qui sera inséré dans le header de votre page HTML : vous devez saisir du contenu compatible HTML. Typiquement, des balises META, SCRIPT ou STYLE.
