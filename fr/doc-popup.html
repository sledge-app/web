Configurer une vue surgissante

Afficher une vue en popup

Il est possible d’afficher une vue depuis une autre sans pour antant sortir de la page courante : pour cela on utilise les vues en popup (surgissante). Voici la marche à suivre...

Salut

1. Créer deux vues.

Pour l’exercice, nous allons créer deux vues :

  • Une vue "Nouvelle Vue" dans laquelle nous ajoutons un composant « Fiche lien » (tout composant de navigation peut aussi convenir).
  • Une vue « carte » dans laquelle nous ajoutons un composant « Carte avec titre ».

2. Créer une page.

Créez une nouvelle page « carte » dans laquelle vous allez glisser la vue « Nouvelle vue ».

3. Définir le lien

Nous allons maintenant dire à la vue « Nouvelle vue » que nous voulons que le lien provoque l’ouverture de la vue Carte, sans changer la page courante.


  • Depuis « Nouvelle vue » sélectionnez le composant « Fiche Lien » puis ouvrez le panneau de propriétés, onglet "Général".
  • Dans le menu « Choisir une action », sélectionnez « Ouvrir une vue »
  • Un menu « Choisir une vue » s’affiche alors. Choisissez votre vue «Carte».

4. Publier le projet.

Une fois votre projet publié, naviguez jusqu’à la page « carte » et cliquez sur votre lien. Cela aura pour effet d’ouvrir la vue « Carte » au dessus de la page courante, sans pour autant la quitter. Une croix en haut à droite vous permet de fermer le panneau et de revenir à votre page.

Vous pouvez ainsi ouvrir n’importe quel contenu en popup.
