Ajouter une carte

Ajouter une carte à votre site !

Seulement avec SledPlus.

Salut

Sled propose deux composants qui permettent d’afficher des cartes Google Map. Carte et Carte avec Titre, le second ajoutant simplement un titre au dessus de la carte pour plus de commodité.

Salut

1. Ajouter le composant.

Ajoutez le composant Carte depuis le catalogue de composants pour qu’il soit disponible dans votre espace de travail.

Salut

2. Déposer le composant dans la vue.

Depuis le menu latéral, glissez la carte dans la vue.

Salut

3. Coller le code de la carte.

Ouvrez le panneau de propriétés et collez le code HTML de votre carte Google Map. Il doit s’agir d’une balise iFrame que vous aurez trouvé depuis l’option de partage "intégrer" sur Google Map.

Vous devez paramètrer une carte par langue.
