Ajouter une vidéo

Ajouter une vidéo à mon site.

Sled permet d’ajouter des vidéos à votre site en utilisant l’option de partage embed (embarquée) de votre plateforme vidéo favorite (Youtube, Vimeo, Dailymotion…). Vous ne pouvez pas ajouter un fichier video au projet, pour des raisons de place et de performance d’hébergement.

Salut

1. Ajouter le composant Vidéo.

Depuis le catalogue de composants, ajoutez le composant Vidéo à votre projet.

Salut

2. Glisser le composant.

Allez sur la vue de votre choix puis glissez le composant depuis la liste de composants.

Salut

3. Définir une taille.

Dans le panneau de propriétés de votre composant, vous devez définir la taille (ratio) de votre vidéo. La plus courante est 16:9 (Écran large) mais 9:16 conviendra par exemple si vous avez filmé à la verticale (portrait) avec votre téléphone.

Salut

4. Coller le code.

Vous devez trouver sur votre plateforme de vidéo (Youtube, Vimeo, Dailymotion) l’option de partage Embed (embarquée) qui vous permet de copier un morceau de code html, contenant une balise iFrame. Elle se trouve en général dans un bouton ‘partage’ (Sharing). Collez le code généré dans le panneau de propriétés.

Voici la liste des documentations de quelques plateformes:

  • Youtube
  • Vimeo
  • Dailymotion
Salut
